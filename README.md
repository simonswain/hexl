# Hexl

Generate or display a hex dump

## Quickstart

```
npm install hexl
```

```
const Hexl = require('hexl');

// return the hexdump as a string
Hexl.hexl(<buffer|string>, [caption]);

// output the hexdump directly
Hexl.log(<buffer|string>, [caption]);
```
